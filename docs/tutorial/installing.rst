Installing
==========

The package is found from `Pypi <https://pypi.org/project/redengine/>`_. Just Pip install 
it as:

.. code-block:: console

    pip install redengine

The minimum dependencies are installed by default.
However, some built-in tasks, conditions and 
functionalities require additional packages.
